## üéÆ Running Your First Game
Now that you have DOSBox installed, here‚Äôs how to run a DOS game:

1. **Locate Your Game Files**
Ensure your game files are available on your computer. You can often find these files online or extract them from old discs.

2. **Mount the Game Directory**
In DOSBox, you need to tell it where your game files are. Use the following command:
```
mount C /path/to/your/game
```
Replace `/path/to/your/game` with the actual path where your game files are stored.

3. **Changing to the Game Directory**
Type `C:` and hit Enter to switch to the mounted directory.

4. **Launch the Game**
Find the executable file for your game (often named `GAME.EXE` or similar) and type its name, then hit Enter. Enjoy your game!
